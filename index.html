Claude Desktop App

แอปพลิเคชันสำหรับ Windows และ macOS ใช้ฟรี แต่ให้ประสิทธิภาพดีกว่าเบราว์เซอร์

ฟรี
ดาวน์โหลดและติดตั้ง
1
ดาวน์โหลดไปที่ claude.ai/download หรือ anthropic.com/claude/download
2
ติดตั้งWindows: ไฟล์ .exe/.msix | Mac: ไฟล์ .dmg
3
ล็อกอินใช้บัญชี claude.ai เดียวกัน ไม่ต้องสมัครใหม่
4
ตั้ง ShortcutAlt+Space (Windows) หรือ ⌥+Space (Mac) เรียก Claude ได้ทันที
ข้อดีเหนือกว่าเบราว์เซอร์
⚡

เร็วกว่า 3-4 เท่า

เปิดแอปพร้อมใช้ใน 3 วินาที เทียบกับเบราว์เซอร์ที่ใช้ 10-12 วินาที

🎯

Global Shortcut

กด Alt+Space จากแอปไหนก็ได้ เรียก Claude ขึ้นมาทันทีโดยไม่ต้องสลับหน้าต่าง

💾

ใช้ RAM น้อยกว่า 5 เท่า

แอปใช้ RAM เพียง ~200-400MB ส่วนเบราว์เซอร์อาจสูงถึง 1.2-2GB ต่อแท็บ

🔌

MCP Extensions

ติดตั้ง Extensions ด้วยคลิกเดียว เพื่อเชื่อมต่อกับเครื่องมือต่างๆ แบบ Native

📂

Drag & Drop ไฟล์

ลากไฟล์จากเครื่องมาวางในแชทได้โดยตรง สะดวกกว่าการอัปโหลดในเบราว์เซอร์

🛡️

ไม่มีปัญหา Tab Refresh

แท็บเบราว์เซอร์รีเฟรชทำให้สูญเสียบทสนทนา แต่แอป Desktop ไม่มีปัญหานี้

Browser vs Desktop — เลือกอะไรดี?
🌐
ใช้เบราว์เซอร์เมื่อ...

ใช้งานเบาๆ เป็นครั้งคราว, ต้องการแชร์ลิงก์บทสนทนา, ใช้หลายอุปกรณ์, ไม่อยากติดตั้งโปรแกรม

🖥️
ใช้ Desktop App เมื่อ...

ใช้งาน Claude ทุกวัน, ทำงานกับไฟล์หนัก, ต้องการ Global Shortcut, ต้องการรัน Cowork, อยากประหยัด RAM และต้องการความเร็ว

🤝

Claude Cowork

AI Agent สำหรับคนที่ไม่ใช่นักพัฒนา ทำงานหลายขั้นตอนอัตโนมัติในเครื่องของคุณ

Pro+ เท่านั้น
🤖
Cowork คืออะไรกันแน่?

แทนที่จะคุยกับ Claude ทีละข้อความ Cowork ให้ Claude ทำงานซับซ้อนหลายขั้นตอนแบบอัตโนมัติในเครื่องของคุณ เหมือนมีผู้ช่วยนั่งทำงานให้ บอกแค่ "ต้องการอะไร" แล้วกลับมาดูงานที่เสร็จแล้ว

ความต้องการระบบ
💻

Windows หรือ macOS

ต้องใช้ Claude Desktop App (Windows 10+ หรือ macOS) ไม่รองรับเว็บหรือมือถือ

💳

แผน Pro ($20/เดือน) ขึ้นไป

ใช้ได้กับ Pro, Max, Team, Enterprise เท่านั้น ผู้ใช้ฟรีใช้ไม่ได้

วิธีเริ่มใช้ Cowork
1
เปิด Claude Desktop Appต้องเป็น Desktop App เท่านั้น
2
คลิกแท็บ "Cowork" หรือ "Tasks"อยู่ถัดจากแท็บ Chat ด้านบน
3
เลือกโฟลเดอร์ที่ต้องการให้ Claude เข้าถึงลากโฟลเดอร์มาวาง หรือคลิกเลือก
4
อธิบายงานที่ต้องการบอกผลลัพธ์ที่ต้องการ Claude จะวางแผนและทำให้
5
ดูหรือปล่อยให้ทำงานติดตาม progress ได้ตลอด หรือไปทำอย่างอื่นแล้วค่อยกลับมา
ตัวอย่างงานที่ Cowork ทำได้
📊

วิเคราะห์ข้อมูลจำนวนมาก

ประมวลผล Excel ขนาดใหญ่, สรุปข้อมูล, สร้างรายงาน Word อัตโนมัติ

คำสั่งตัวอย่าง

"โฟลเดอร์นี้มีใบเสร็จ 50 ไฟล์ ดึงวันที่และยอดเงินแล้วรวมใส่ Excel ให้ด้วย"

📁

จัดการและจัดเรียงไฟล์

จัดระเบียบไฟล์ตามประเภท, เปลี่ยนชื่อไฟล์เป็นระบบ, ย้ายไฟล์

คำสั่งตัวอย่าง

"จัดเรียงรูปภาพในโฟลเดอร์ Downloads ตามปีและเดือน"

🔬

วิจัยอัตโนมัติ

ค้นหาข้อมูลออนไลน์ วิเคราะห์ และสรุปรายงานโดยอัตโนมัติ

คำสั่งตัวอย่าง

"วิจัยคู่แข่ง 5 ราย เปรียบเทียบราคาและฟีเจอร์ สรุปเป็น Word"

⏰

ตั้งเวลาทำงานอัตโนมัติ

ใช้ /schedule ให้ Cowork ทำงานซ้ำๆ อัตโนมัติตามเวลาที่กำหนด

วิธีใช้

พิมพ์ /schedule แล้วระบุงานและเวลา

🛡️
ความปลอดภัยของ Cowork

Cowork ทำงานใน Virtual Machine (VM) ที่แยกออกมา ไฟล์ของคุณไม่ถูกส่งออกไปอินเทอร์เน็ต และก่อนลบไฟล์ใดๆ จะต้องขอการยืนยันจากคุณทุกครั้ง

Cowork vs Claude Code ต่างกันอย่างไร?
🤝

Cowork — สำหรับทุกคน

Interface กราฟิก ใช้ง่าย ปลอดภัยกว่า ทำงานใน VM เหมาะสำหรับนักธุรกิจ นักวิเคราะห์ ที่ไม่ถนัด Terminal

⌨️

Claude Code — สำหรับนักพัฒนา

Terminal-based ควบคุมได้เต็มรูปแบบ Git-aware รัน Test ได้ เหมาะสำหรับ Developer ที่ต้องการพลังสูงสุด

⌨️

Claude Code

เครื่องมือ Agentic Coding สำหรับนักพัฒนา รันใน Terminal/IDE ด้วยพลังสูงสุด

สำหรับนักพัฒนา
💡
Claude Code คืออะไร?

AI ที่เข้าใจโค้ดทั้งโปรเจกต์ (สูงสุด 200,000 tokens) แก้บัค, สร้างฟีเจอร์, รัน test, จัดการ Git ได้จาก Terminal โดยตรง เหมือนมี Senior Developer นั่งคู่กัน

ติดตั้ง Claude Code
# ติดตั้งผ่าน npm npm install -g @anthropic-ai/claude-code # เริ่มใช้งาน claude # หรือในโฟลเดอร์โปรเจกต์ cd myproject && claude
⚠️
ความต้องการ

Node.js 18+ | Windows 10+ (ต้องมี Git for Windows) | macOS 12+ | ต้องการ Claude subscription

ฟีเจอร์หลักของ Claude Code
🧠

เข้าใจ Codebase ทั้งโปรเจกต์

รองรับ context สูงสุด 200,000 tokens สามารถเข้าใจโครงสร้างโค้ดทั้งโปรเจกต์พร้อมกัน

เทคนิคขั้นเทพ

วิธีที่ทำให้คุณใช้ Claude ได้ดีกว่าคนทั่วไป — ตั้งแต่พื้นฐานถึง Expert

🎯 เทคนิคการเขียน Prompt ที่ดี
1️⃣
บอก Role + Context + งาน

แทนที่จะพูดสั้นๆ ให้บอกว่าคุณเป็นใคร ทำไมถึงต้องการ และต้องการอะไรกันแน่
❌ "เขียนอีเมล"
✅ "ฉันเป็น Sales Manager ต้องการอีเมลขอโทษลูกค้า VIP ที่ส่งสินค้าล่าช้า 3 วัน โทนทางการแต่อบอุ่น ความยาวไม่เกิน 150 คำ"

2️⃣
ใช้ Format Tag เพื่อควบคุม Output

บอก Claude ว่าต้องการ output รูปแบบไหน
"ตอบเป็นข้อๆ ไม่เกิน 5 ข้อ" | "ตอบเป็นตาราง" | "ตอบสั้นๆ ใน 2 ประโยค" | "ใช้รูปแบบ JSON"

3️⃣
Chain of Thought — ขอให้คิดก่อนตอบ

สำหรับคำถามซับซ้อน พิมพ์เพิ่มว่า "คิดทีละขั้นตอนก่อนตอบ" หรือ "อธิบายวิธีคิดก่อน" — ผลลัพธ์จะแม่นยำขึ้นมาก

4️⃣
ขอตัวเลือกหลายแบบ

แทนที่จะรอคำตอบเดียว ลองขอ 3 แบบแล้วเลือก
"ให้ 3 ตัวเลือกในการตอบปัญหานี้ พร้อม trade-off ของแต่ละแบบ"

5️⃣
ใช้ Projects + Custom Instructions

สร้าง Project และตั้ง Instructions เช่น "ฉันเป็นนักการตลาด ตอบเสมอเป็นภาษาไทย ใช้คำง่าย หลีกเลี่ยงศัพท์เทคนิค" — Claude จะจำทุกครั้งในโปรเจกต์นั้น

6️⃣
ให้ตัวอย่างที่ต้องการ (Few-Shot)

ถ้าต้องการรูปแบบเฉพาะ ให้ตัวอย่างก่อน
"ฉันต้องการรูปแบบนี้: [ตัวอย่าง] ตอนนี้ทำให้ฉันด้วยข้อมูลนี้: [ข้อมูล]"

7️⃣
ขอวิจารณ์งานตัวเองก่อน

หลังได้คำตอบแล้วลองพิมพ์ว่า
"ทบทวนคำตอบของตัวเอง มีจุดไหนที่อาจผิดหรือปรับปรุงได้บ้าง?"

8️⃣
Extended Thinking — เปิดโหมดคิดลึก

สำหรับปัญหา logic ซับซ้อน, คณิตศาสตร์, หรือการวางกลยุทธ์ เปิด Extended Thinking ใน Settings — Claude จะ "คิด" ก่อนตอบ ผลลัพธ์แม่นยำขึ้นมาก
